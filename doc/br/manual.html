Introdução

O LuaEclipse é uma coleção de plug-ins para a plataforma Eclipse. Esta ferramenta se propõem a facilitar o desenvolvimento de aplicações que utilizem a linguagem Lua. Em um primeiro momento de forma standalone, e em seguida de forma integrada com uma variedade de outras linguagens.

Os plug-ins são extensíveis e podem ser utilizados como base para outras ferramentas. A pesar desta possibilidade, esta seção resume-se em um manual de utilização do software e não de extensão do mesmo.

LuaEclipse é um software livre e usa a mesma licença do Lua 5.0.

Instalação

O processo de instalação do plug-in é muito simples, mas necessita de alguns pr&ecute;-requisitos. O primeiro deles é que o usuário já possui instalada a versão 3.2 da plataforma Eclipse. O Eclipse pode ser baixado no endereço www.eclipse.org. Consequentemente será necessário a instalação prévia do SDK de Java para executar o Eclipse.

Existem duas formas de instalar o LuaEclipse.

  • A partir do download manual:
    • Faça o download da última versão do LuaEclipse no site LuaForge. Para instalar no Eclipse, basta descompactar o arquivo zip na pasta raiz da instalação do eclipse, por exemplo: "/usr/local/eclipse". Feito isto, inicialize\reinicialize o Eclipse e a instalaç&atlde;o está concluída.
      Para verificar se a instalação foi concluída com sucesso, vá até a opção de menu Help->About Eclipse SDK e selecione a opção Plug-ins Details e verifique se o LuaEclipse esta instalado corretamente. Veja a figura abaixo:
